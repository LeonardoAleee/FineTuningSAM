# Fine-Tuning do Segment Anything Model (SAM) — Implementação e Resultados

Este repositório contém a implementação do fine-tuning do Segment Anything Model (SAM) aplicada a uma tarefa específica de segmentação, desenvolvida para o Assignment 4 da disciplina de Deep Learning. O objetivo é adaptar o SAM para um dataset escolhido e avaliar o impacto do ajuste no desempenho do modelo.

O vídeo de apresentação está disponível no seguinte link: https://drive.google.com/file/d/1PmEQhmUqJUkfxKCNil298FyWpUyZc1F3/view?usp=sharing.
